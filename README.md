# Tracker
This is an attempt to make a Tracker for an RC plane
I will be using the most commonly available 9DOF IMU chip around me which is ICM20948
controlled by a MCU in our case ESP32
later LORA modules will be added to remotely communicate and get data from the loaded IMU 
Throughout this project I will be learning ESP IDF and documenting my experience, errors and how i resolved them.
